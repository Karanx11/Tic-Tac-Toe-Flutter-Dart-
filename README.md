# Tic Tac Toe Flutter App

This is a simple Tic Tac Toe game built with Flutter that runs smoothly on mobile devices (iOS & Android). It’s designed as a two-player game on the same device with an elegant UI, animations, and all the core features you’d expect.

---

## 🎯 Task

Build a simple Tic‑Tac‑Toe app that runs on mobile devices (iOS & Android) using Flutter. You may choose any state‑management approach (e.g., setState, Provider, BLoC).

- Two‑player game on the same device  
- 3×3 grid UI, display current player (“X” or “O”)  
- Detect and indicate a win or a draw  
- “Restart” button to reset the board  
- Simple animations or highlights when a player wins  
- Any other creative feature to distinguish your app

Deliverables:

- Demo Recording: A quick screen recording (30–60 sec) showing gameplay: placing X’s and O’s, win/draw detection, and restart.  
- Source Code: A GitHub repo link with your Flutter project

---

## 📱 Features

- **Two-player mode:** Play on the same device taking turns between “X” and “O”.  
- **3×3 grid:** Classic Tic Tac Toe layout with smooth UI and a subtle glassy, blurred background effect.  
- **Current player display:** Shows whose turn it is.  
- **Win/draw detection:** Automatically detects a winner or draw and displays dialogs accordingly.  
- **Scoreboard:** Keeps track of wins for both players.  
- **Animations:** Animated scaling effect on tapping cells and a colorful fireworks celebration on winning.  
- **Restart & Reset Scoreboard:** Buttons to restart the current game or reset all scores.  
- **Clean UI:** Dark gradient background with translucent grid cells and glowing effects for winning cells.

---

## 🛠️ Tech Stack

- Flutter & Dart  
- State management: `setState` (simple and effective for this app)  
- Custom animations with `AnimationController`  
- UI built with `GridView`, `AnimatedScale`, and `BackdropFilter` for blur effects
